# Mass Media Master

Скрипт-автоматизация для SA-MP (San Andreas Multiplayer), написанный на Lua под мод [MoonLoader](https://www.blast.hk/threads/13305/). Предназначен для игроков ролевых (Role-Play) серверов, выполняющих должность в медиа-структуре сервера: автоматизирует часто повторяющиеся ролевые сценарии, ведение лекций, собеседований и работу с типовыми объявлениями.

## Возможности

- **Графический интерфейс на ImGui** — окно настроек с переключателями поведения скрипта, вызывается по горячей клавише.
- **Автоматизация ролевых сценариев** — заготовленные цепочки команд `/me`, `/do`, `/r` для типовых ситуаций: интервью, собеседования, лекции по разным темам, объявления в эфире.
- **Кастомные шаблоны ("бланки")** — пользовательские текстовые заготовки хранятся в `blanks.json` и подгружаются при старте; можно редактировать без правки кода.
- **Гибкие горячие клавиши** — привязки хранятся в `settings.bind` и переназначаются через интерфейс, используется библиотека `rkeys`.
- **Персистентные настройки** — конфигурация сохраняется через `inicfg` (пол персонажа, структура организации, задержки между репликами, тег и т.д.).
- **Полезные команды в чат:**
  - `/me`, `/do`, `/r`, `/f` — с автоматической капитализацией первой буквы и расстановкой знаков препинания;
  - `/proc` — быстрое меню действий над игроком (приём/увольнение/выговор/смена должности);
  - `/cfind` — поиск игроков без активного RP-статуса среди онлайна с групповыми действиями;
  - набор команд для новостных RP-механик (`/testgnews`, `/gognews`, `/gognewstime`).
- **Поддержка кириллицы (CP1251)** — собственная реализация приведения регистра для русских букв, так как стандартные функции Lua `string.upper/lower` с этой кодировкой не работают.

## Технологии

- **Lua** (диалект, используемый MoonLoader)
- **MoonLoader** — среда выполнения скриптов для SA-MP
- Библиотеки экосистемы MoonLoader: `imgui`, `inicfg`, `encoding`, `memory`, `lib.samp.events`, `vkeys`, `rkeys`, `game.keys`, `faIcons`

## Требования

- SA-MP клиент с установленным [MoonLoader](https://www.blast.hk/threads/13305/)
- Стандартный набор библиотек MoonLoader (входят в дистрибутив или ставятся отдельно): `imgui`, `inicfg`, `rkeys`

## Установка

1. Скачайте файл `Mass_Media_Master.lua`.
2. Поместите его в папку `moonloader` вашего SA-MP клиента.
3. Запустите игру — при первом старте скрипт создаст рабочую директорию и файлы конфигурации (`blanks.json`, `settings.bind`) со значениями по умолчанию.

## Использование

После запуска скрипт выводит в чат подсказку с текущей горячей клавишей вызова меню. Через интерфейс доступны разделы: работа с эфиром, лекции, ролевые сценарии, структура организации, настройки.
